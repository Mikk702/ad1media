AWS Demo - Version 1

This page demonstrates the functionality of a basic web application using several key AWS services. It serves as a proof-of-concept and foundation for future development.

Key Features Demonstrated:

  • Static Website Hosting: The website files (HTML, CSS, JavaScript) are hosted on Amazon S3 and served via Amazon CloudFront CDN.
  • Serverless Compute: Dynamic elements and API logic are handled by AWS Lambda functions.
  • Relational Database: Data is stored and retrieved from an Amazon RDS for MySQL database.
  • Identity and Access Management: User authentication and authorization are managed by Amazon Cognito.
  • File Storage: User-uploaded files are stored securely in Amazon S3.
  • Infrastructure as Code (YAML): A YAML configuration file representing the basic architecture is generated and displayed.

AWS Services Used:

  • Compute:

    Amazon EC2

    Amazon Elastic Compute Cloud (EC2) provides scalable virtual servers in the cloud (instances). You can customize these instances with your choice of operating systems, software, and configurations.

    AWS Lambda

    AWS Lambda lets you run code without provisioning or managing servers. You pay only for the compute time you consume.

  • Storage:

    Amazon S3

    Amazon Simple Storage Service (S3) is object storage built to store and retrieve any amount of data at any time.

  • Database:

    Amazon RDS for MySQL

    Amazon Relational Database Service (RDS) makes it easy to set up, operate, and scale a relational database in the cloud.

  • Security:

    AWS IAM

    AWS Identity and Access Management (IAM) enables you to manage access to AWS services and resources securely.

    Amazon Cognito

    Amazon Cognito lets you add user sign-up, sign-in, and access control to your web and mobile apps quickly and easily.
