AWS Demo - Version 1

This page demonstrates the functionality of a basic web application using several key AWS services. It serves as a proof-of-concept and foundation for future development.

Key Features Demonstrated:

  • Static Website Hosting: The website files (HTML, CSS, JavaScript) are hosted on Amazon S3 and served via Amazon CloudFront CDN.
  • Serverless Compute: Dynamic elements and API logic are handled by AWS Lambda functions.
  • Relational Database: Data is stored and retrieved from an Amazon RDS for MySQL database.
  • Identity and Access Management: User authentication and authorization are managed by Amazon Cognito.
  • File Storage: User-uploaded files are stored securely in Amazon S3.
  • Infrastructure as Code (YAML): A YAML configuration file representing the basic architecture is generated and displayed.

AWS Services Used:

  • Compute: Amazon EC2 (example), AWS Lambda
  • Storage: Amazon S3
  • Database: Amazon RDS for MySQL
  • Security: AWS IAM, Amazon Cognito
